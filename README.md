Weeb — Landing Page Deployment
Проєкт адаптивної верстки лендингу "Weeb", виконаний за макетом з Figma. Сайт містить головний екран, блоки з послугами, відгуками та майбутніми івентами.

🚀 Технології
HTML5 — семантична розмітка.
CSS3 — BEM-методологія, Flexbox, Grid Layout, анімації.
JavaScript — функціонал бургер-меню та інтерактивних елементів.
Google Fonts — підключено шрифт Roboto.

📋 Особливості реалізації
Pixel Perfect: Максимальна відповідність макету Figma.
Adaptive Design: Повна адаптивність під мобільні пристрої та планшети.
Animations: Окремий файл animations.css для плавних ефектів появи блоків.
Architecture: Модульна структура CSS (розділено на компоненти).

📁 Структура проєкту
Plaintext
├── index.html          # Головна сторінка
├── css/
│   ├── style.css       # Основні стилі
│   ├── reset.css       # Скидання стандартних стилів
│   ├── components.css  # Стилі окремих компонентів 
│   ├── media.css       # Адаптив під різні екрани
│   └── animations.css  # CSS-анімації
├── js/
│   └── script.js       # Логіка меню та взаємодії
├── img/                # Оптимізовані SVG та зображення
└── README.md
